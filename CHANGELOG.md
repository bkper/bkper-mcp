# Changelog

Public changes to the hosted Bkper MCP service.

## June 2026

- Added Bkper app metadata tools to help users inspect installed app context.
- Added Bkper Book, query, and chart links in relevant tool outputs so users can open the related Bkper context.
- Added OpenAccountants tax-rule and accountant-referral lookups by jurisdiction, without sending private Book data to OpenAccountants.
- Improved tax-preparation and accountant-handoff guidance for provisional workpapers and professional review.
- Documented the canonical MCP connector URL: `https://mcp.bkper.app/mcp`.
- Improved compatibility with Claude connectors.
- Added MCP discovery links in Bkper.
- Improved reliability for long-running connector sessions.
